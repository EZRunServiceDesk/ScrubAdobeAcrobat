# ScrubAdobeAcrobat
Removed x86 or x64 of Adobe Acrobat 


Edit ScrubOffAdobeAcrobat.ps1 

Change from True to false for any of the ones you don't want 

# x86 Adobe Reader
$Reader32PS = "True"
# x64 Adobe Reader
$Reader64PS = "True"
# x86 Adobe Acrobat
$Acrobat32PS = "True"
# x64 Adobe Acrobat
$Acrobat64PS = "True"
